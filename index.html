# Guía de Despliegue - Plan Misional EP ¡Felicidades! Tu aplicación está lista para ser publicada en internet. Sigue estos pasos en tu terminal. ## Prerrequisitos Asegúrate de estar en la carpeta raíz del proyecto (donde están estos archivos). ## Paso 1: Instalar herramientas de Firebase (solo una vez) Si no tienes las herramientas de Firebase instaladas, ejecuta: ```bash npm install -g firebase-tools ``` ## Paso 2: Construir la aplicación Esto genera la carpeta `dist` con tu aplicación optimizada para producción. ```bash npm install npm run build ``` *(Nota: Si usas Vite, esto creará una carpeta `dist`. Si usas Create React App, creará `build`. Si se crea `build`, cambia "public": "dist" por "public": "build" en el archivo firebase.json).* ## Paso 3: Iniciar sesión en Firebase ```bash firebase login ``` Sigue las instrucciones en el navegador para autorizar el acceso con tu cuenta de Google. ## Paso 4: Desplegar Este comando subirá tu aplicación y las reglas de base de datos a los servidores de Google. ```bash firebase deploy ``` --- ## ¡Listo! Al finalizar, la terminal te mostrará una **Hosting URL** (ejemplo: `https://plan-misional-ep.web.app`). Esa es la dirección que compartirás con los líderes y misioneros.
